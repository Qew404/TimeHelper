#  TimeHelper

> Простое и ненавязчивое приложение для отслеживания рабочего времени на Windows.  
> *Ваш личный таймер, который знает, когда вы ушли с рабочего места.*

---

##  Ключевые возможности

 **Гибкие пресеты**  
Выбирайте рабочую смену от **2 до 12 часов** одним кликом — никаких ручных настроек!

 **Умная пауза**  
- Ручное управление: кнопка «Пауза/Продолжить» в интерфейсе  
- **Автоматическая пауза** при блокировке системы (`Win + L`)  
- **Мгновенное возобновление** после разблокировки компьютера  

 **Чистый интерфейс**  
Минималистичный дизайн без отвлекающих элементов. Только суть: таймер, прогресс и управление.

 **Работает в фоне**  
Не мешает работе, не нагружает систему, всегда готов к использованию.

---

##  Как использовать

1. Запустите приложение  
2. Выберите длительность рабочего дня из списка:  
   `2ч` • `4ч` • `6ч` • `8ч` • `10ч` • `12ч`  
3. Нажмите **«Старт»** — таймер пошёл!  
4. Уходите с рабочего места? Просто нажмите `Win + L` — таймер сам встанет на паузу.  
5. Вернулись? Разблокируйте ПК — отсчёт продолжится автоматически.  

 *Никаких настроек, плагинов или регистрации. Просто работает.*
